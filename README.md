
# 🇫🇷 French Cheatsheet  
A concise, beginner‑friendly reference for essential French grammar, vocabulary, and everyday phrases.  
Designed for English speakers learning French and wanting a quick, practical guide.

---

## 📘 Overview  
This repository contains a simple **HTML cheatsheet** that summarizes the most useful elements of French for daily conversation:

- Common verbs and conjugations  
- Useful sentence structures  
- Everyday vocabulary  
- Practical phrases for travel and conversation  
- Quick grammar reminders  

The goal is to provide a **fast, accessible reference** you can open anytime while learning or practicing French.

---

## 📂 Repository Structure

| File | Description |
|------|-------------|
| `index.html` | The main French cheatsheet page (open in any browser). |
| `README.md` | Documentation for the project. |

## 🚀 How to Use  
1. Clone or download the repository:  
   ```bash
   git clone https://github.com/gireesh-kokate/french-cheatsheet.git
   ```
2. Open `index.html` in your browser.  
3. Use it as a quick reference while studying or practicing French.
